## 1. 배경 / 문제

## 2. 사용자가 선택한 방향(결정)
- 계층화 방식: **마스터 트래킹 이슈 1장**(네이티브 sub-issue/라벨 대신).
- 정리 범위: **전체를 상위 로드맵 1장**으로.
- 중복 처리: **건별 확인 후 결정**.
  - #60을 "폰트 탐색 경험" **에픽 부모**로 삼고 #55-#56-#57-#54를 하위 링크(닫지 않음).
  - #56은 #28과 시드 중복 → **시드는 #28로 일원화, #56은 컬렉션 페이지 고도화로 범위 축소**.
- 라벨: `urgent` `priority: high/medium/low` `epic` 신설 후 부여.
