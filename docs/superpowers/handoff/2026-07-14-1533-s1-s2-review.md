### 사용자 제약-불가침 결정 (반드시 준수)

🔴 **반드시 (must)** — 이전 세션들에서 확정된 결정, 뒤집지 말 것:
- 스타일은 **CSS Modules + CSS 변수 토큰** (Tailwind 미사용)
- 배포는 **정적 사이트 생성** (`next.config.ts`의 `output: 'export'`) — SSG 우선
- 폰트는 앱에 self-host (next/font + Pretendard)
- 화면 데이터는 현재 목업 → **추후 파이프라인 실데이터로 교체 예정** (교체가 S1/S2의 핵심)
- 비교 화면은 폰트 슬롯 3개 고정 + select 교체
- OG 이미지는 satori에 한글 폰트(Pretendard-Bold.otf) 임베드 (미임베드 시 한글 깨짐)
- Git 워크플로: develop에서 작업 → develop→main. develop 브랜치 삭제 안 함. **main과 develop은 PR 머지 커밋 때문에 상시 분기 상태 → develop→main은 fast-forward 불가, `--no-ff` 머지 커밋으로 합침** (지난 세션에서 확인)
- 인프라(기획): 로컬 배치 → **Supabase**(DB/검색/스토리지, 자체 호스팅 계획) → Next.js SSG → Cloudflare CDN
